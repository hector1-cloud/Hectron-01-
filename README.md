# Hectron-01

## Descripción
Hectron-01 es una implementación de inteligencia artificial personal desarrollada en Python utilizando el framework Flet para la interfaz gráfica.

## El Nuevo Códice
Este proyecto busca integrar lógica computacional con narrativa personal, explorando la interacción entre usuario y máquina bajo la arquitectura "Hectron".

## Instalación
1. Clona el repositorio.
2. Instala las dependencias:
   `pip install -r requirements.txt`
3. Ejecuta el sistema:
   `python main.py`
